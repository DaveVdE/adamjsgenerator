Adding a block

Most statements invoke other statements. The if condition statement has a 'then' part and an 'else' part.

While these parts consist of single statements, we can use a BlockStatement to wrap multiple statements in a 'block', which produce the curly braces that we all know and love.

The extension methods that allow you to specify statements to add, like Do(), Then() and Else() allow you to specify a single statement, multiple statements or and IEnumerable of Statement, and the helper method will decide whether to wrap them in a block or not.

var a = JS.Id("a");

return JS.For(null, a.IsGreaterThan(0), a.PostDecrement()).Do(

    JS.Alert(a),

    JS.If(a.IsEqualTo(5)).Then(

        JS.Break()));
