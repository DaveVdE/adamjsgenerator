Implicit Conversions

When you need to supply an Expression, you can always use a simple string object, int, double or boolean.

Also, when an IdentifierExpression is required, you can simply pass a string.

Examples:

Expression s = "Hello, World!";

Expression d = Math.PI;

Expression i = int.MaxValue;

Expression b = true;

IdentifierExpression id = "id";
