Statement Basics

Statements in JavaScript are represented by descendants of the abstract Statement class. They're easily created using the JS static class and extension methods.

Some examples:

  • JS.Break()
  • JS.Do().While()
  • JS.If().Then().Else()
  • JS.Try().Catch().Finally()

Source:

var a = JS.Id("a");

return JS.For(JS.Var(a.AssignWith(10)), a.IsGreaterThan(0), a.PostDecrement())

    .Do(JS.Alert(a));
